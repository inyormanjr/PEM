Help Section

To use the D.E Error Code Lookup application:

  • Enter an error code or description in the search bar.
  • Press the search button or hit enter to initiate the search.
  • Review the results displayed in the accordion below the search bar.
  • Click on an error code to see detailed information.

Error codes are used to identify specific issues related to specimen collection, labelling, handling, and more. Here are some examples:

  • MCOL: Missing Specimen or incomplete Collection - Hepatitis serology testing requested but no blood collected.
  • WSCO: Wrong Specimen Collected (Wrong container) - Von Willebrand Screen (VWS) requested but SST (Gold) collected.
  • ETCO: Excessive Tubes Collected - x3 EDTA tubes collected for Full Blood Count (FBC).
  • ISSC: Insufficient Sample Collected - Citrate tube underfilled for INR.
  • INSL: Incorrect Specimen or Request form Labelling - Request for John Smith, specimen labelled for John Citizen.

Here are some common issues you may encounter and their solutions:

  • Issue: No results found.
    Solution: Ensure that the error code or description is entered correctly. Try using different keywords.
  • Issue: Error code details are not clear.
    Solution: Refer to the detailed descriptions and examples provided in the help section.
  • Issue: Unable to copy example text.
    Solution: Ensure you are clicking the copy button next to the example text. A success message should appear if the text is copied successfully.
